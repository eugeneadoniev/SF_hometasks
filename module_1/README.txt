Project 1. Who Wants to Become a Film Millionaire?
Exploratory data analysis in Python Pandas.
Objective of the project:
Analyze the data based on the IMBD site dataset (database and website of the film distribution company).
Initial data: CSV file. I have loaded it into Pandas Dataframe.
Project stages:
1. Preprocessing - clearing data, working with string values.
2. Writing queries to the Dataframe to answer the questions posed (27 queries in total)

Used Python modules:
- numpy
- seaborn
- matplotlib.pyplot
- collections
- itertools

As a result, the necessary information on the input dataset is obtained. The answers are collected in a Python dictionary.

Project made on Jupyter notebook.
